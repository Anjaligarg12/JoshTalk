# Josh Talks AI – Human Evaluation Benchmark for Text-to-Image Models

A comprehensive human evaluation platform designed to compare AI-generated Indian ethnic clothing images across multiple text-to-image models.

---

## Project Overview

This project was developed as part of the Josh Talks AI Product Task.

The objective is to evaluate AI-generated Indian ethnic clothing images using structured human feedback instead of relying only on automated metrics.

Participants compare outputs generated by different AI models and rate them across multiple quality dimensions.

---

## Problem Statement

Modern AI image generation models produce visually impressive images, but their usefulness for Indian users depends on factors such as:

- Realism
- Cultural Accuracy
- Visual Appeal
- Fabric Details
- E-commerce Readiness

This evaluation platform helps compare different AI models through human judgment.

---

## AI Models Compared

- OpenAI – GPT Image 1

## Evaluation Criteria

Each generated image is evaluated on:

⭐ Realism

⭐ Cultural Accuracy

⭐ Visual Appeal

⭐ Fabric Details

⭐ E-commerce Readiness

Participants also select:

- Best Image
- Best Overall Model

## Evaluation Workflow

1. Participant opens Evaluation Dashboard.

2. Images generated from three AI models are displayed.

3. Each image is rated.

4. Best image is selected.

5. Overall model ranking is provided.

6. Feedback is submitted.

7. Responses are stored for analysis.
