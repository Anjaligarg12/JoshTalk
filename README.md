# Project Overview

This project evaluates AI-generated Indian ethnic clothing images through human judgment instead of relying only on automated evaluation metrics.

Participants compare outputs generated by multiple AI models and rate them using predefined evaluation criteria. The collected responses help identify which model performs best for Indian fashion use cases.

---

# Objectives

- Compare multiple Text-to-Image AI models.
- Collect structured human feedback.
- Evaluate model performance for Indian ethnic clothing.
- Identify the most preferred AI model.
- Build a scalable evaluation workflow.

# Evaluation Criteria

Each generated image is evaluated on:

- ⭐ Realism
- ⭐ Cultural Accuracy
- ⭐ Visual Appeal
- ⭐ Fabric Details
- ⭐ E-commerce Readiness

Participants also select:

- Best Image
- Best Overall Model
- Final Feedback

# Evaluation Workflow

1. Open the Evaluation Dashboard.
2. Review AI-generated images.
3. Rate each image on five quality metrics.
4. Select the best image.
5. Rank the AI models.
6. Submit feedback.
7. Store responses in Firebase.
8. Analyze results through the dashboard.
